# Jira-Clone-Frontend

## Overview

This is the frontend of the Jira clone.  It provides the User Interface for the jira Dashboard

## Tech Stack

- *Language: JavaScript*
- *Frontend Library: React.js*

## Installation

```bash
git clone https://github.com/hariom2809/Jira-Clone-Frontend.git
cd Jira-Clone-Frontend
```
```bash
npm install
```
```bash
npm run dev
```
